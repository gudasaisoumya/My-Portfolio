Project information

  • Project Name: Quiz Application
  • Role: Single Member
  • Technology Used: HTML, CSS, JavaScript
  • Team Size: 1

Description :

I developed a single page Quiz application, where a student can enter his roll no, course and be able to answer the quiz questions. I also created a quiz with a timer to close the quiz after the allotted time.For loading the quiz questions dynamically use AJAX to fetch contents from a JSON file and render the questions. After clicking the Submit button / timer runs out, I evaluated the quiz answers and print the result on the same page.
